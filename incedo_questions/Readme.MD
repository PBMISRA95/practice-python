# Incedo Interview Questions - Python Practice

A good collection of 30 basic Python programming interview questions and solutions, organised for systematic learning and interview preparation.

## Overview

This directory contains solutions to common Python interview questions covering fundamental to intermediate programming concepts. Questions are organised into logical groups and also include multiple implementations.

## Directory Structure
```
incedo_questions/
├── q1_q10/ # Implementation of Questions 1 - 10
├── q11_q20/ # Implementation of Questions 11 - 20
├── q21_q30/ # Implementation of Questions 21 - 30
├── practice_days/ # Daily Practice sessions organised by month and day
├── questions.txt   # Original list of all 30 questions
```
